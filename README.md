# MotorPH Employee App

This Java application is the initial version of the MotorPH Employee Management System. It was developed as part of our programming course to simulate employee management features such as payroll computation, leave handling, and tax deduction processing.

## 📌 Features

- View and update employee profiles
- Calculate and generate payslips
- Compute tax deductions
- Manage employee leave requests
- Department and admin role functionalities

## 📂 Class Overview

### ✔ EMPLOYEE
Handles employee data and profile management.
### ✔ TAX
Calculates and displays tax details.
### ✔ DEPARTMENT
Manages department and employee lists.
### ✔ ADMIN
Controls employee, payroll, and department functions.
### ✔ LEAVE
Handles employee leave requests and approvals.
### ✔ PAYROLL
Computes gross/net salary and displays payroll info.
